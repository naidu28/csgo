Home

This app will be for players of the video game called Counter-Strike: Global Offensive. It will consist of the multiple of the game's pro teams' stats, as well as include a special page for the user's favourite team, his/her favourite position to play, and their favourite gun to use in the video game. This entire page's info will be based on what the user enters in a form displayed on the main page. This info will be stored in the database. Using Javascript, alerts will be popping up on the page sometimes to show things like teams and players that are popular. Using PHP, information will be collected when the user fills out the form on the main page.
